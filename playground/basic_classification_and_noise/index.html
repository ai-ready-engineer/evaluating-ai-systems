Basic classification + noise

What this is

Two single-label datasets — customer-support tickets routed to one of five categories (multi-class), and movie-review sentiment as POSITIVE/NEGATIVE (binary). For each, the page guides you through three tabs: Explore the data (no classifier yet), then the ML classifier (TF-IDF + Logistic Regression), then the gen-AI classifier (a few-shot LLM). Predictions and ground truth are precomputed, so you can spin freely — no API calls.
