# Synthetic ITSM Tickets

~1K LLM-generated customer-support tickets tagged with ITSM-flavored multi-labels. Built for the course because no public ServiceNow-style dataset exists.

- **Source:** generated by `generate.py` (LLM-driven), committed for reproducibility.
- **License:** course-internal; generated content, no third-party material.
- **Size:** ~1,000 tickets (configurable).
- **Labels:** multi-label, drawn from `[billing, urgent, refund, account, complaint, hardware, software, network, security, login]`. Each ticket can carry multiple tags.
- **Why this dataset:** **controllable skew.** `generate.py` accepts a per-label frequency vector, so we can dial how rare each tag is — perfect for the majority-tag demo and the class-imbalance discussion.

## Generating

```bash
cd datasets/synthetic_itsm
python generate.py --n 1000 --skew default
```

Outputs `data.csv` with columns `ticket_id`, `text`, `labels` (semicolon-joined).

The generator and a fixed seed are committed so the dataset is reproducible end-to-end.
